# ActiveJobNotifier
Uses ActionCable to alert front-end users of finished ActiveJobs

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'active_job_notifier'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install active_job_notifier
```
